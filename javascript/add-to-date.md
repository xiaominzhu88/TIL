# Javascript add to date

- Add a day

Code Example:

```jsx
let date = new Date();
// add a day
date.setDate(date.getDate() + 1);
```

- Add minutes

Code Example:

```jsx
let date = new Date (),
// add 30 minutes
date.setMinutes ( date.getMinutes() + 30 );
```

- Add seconds

Code Example:

```jsx
let date = new Date (),
// add 10 seconds
date.setSeconds( date.getSeconds() + 10 );
```
